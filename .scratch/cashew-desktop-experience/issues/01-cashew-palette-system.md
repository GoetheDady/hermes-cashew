# Cashew Palette system

## Parent

`.scratch/cashew-desktop-experience/PRD.md`

## What to build

Make the app's visual foundation read as Cashew Desktop rather than a generic chat client. The light and midnight atmospheres should share a cohesive Cashew Palette, and common surfaces such as the app background, message area, sidebar, input, buttons, and settings controls should use those tokens consistently.

## Acceptance criteria

- [x] The main app surface uses warm cashew-inspired tokens for background, text, border, accent, and focus states.
- [x] Midnight atmosphere remains warm and low-brightness rather than default gray/dark slate.
- [x] Existing chat, settings, and history surfaces continue to typecheck and render with the updated tokens.
- [x] The palette avoids a one-note beige field by using restrained contrast between cream, shell brown, honey accent, and muted green/terra support tones where needed.
- [x] New or changed functions include JSDoc where applicable.

## Blocked by

None - can start immediately.
